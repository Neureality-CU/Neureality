Workshop Title:

Building neurogames with the Unicorn Unity Interface

Date and Time:

Saturday, March 2, 11AM - 1PM, EST

Duration:

2 hours

Workshop Overview:

The Unicorn Unity Interface is targeted at game developers and designers, however anyone with interest in BCI can benefit from it. In fact, you can save and stream raw data and classification outputs from your application for later analysis and visualizations, an interesting feature for researchers and data scientists interested in serious games and experimental protocol. Advanced users have a wide plethora of parameters to tweak to improve the performance of their BCI and the more ambitious can challenge themselves building multiplayer experiences. Thanks to Unity cross-platform compatibility, the Unicorn Unity Interface is now compatible with VR headsets and Meta Quest enabling seamless VR-BCI applications.

Preparation Requirements:

The interface is tested and compatible with Unity 2022.3.11f. Superior minor releases of Unity are very likely to be compatible. Major releases are untested, and we cannot guarantee compatibility at this stage. The right version of Unity can be downloaded within the Unity Hub. Additionally, we recommend Visual Studio Code as IDE with the Unity extension. Instructions for setup can be found here. The Unicorn Unity Interface requires license activation through the Unicorn Hybrid Black Suite, both will be provided the day of the event. More information on the tools available in the suite can be found here.
