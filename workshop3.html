Workshop Title:

Use PhysioLabXR to Build BCIs

Date and Time:

Saturday, March 2, 3 - 4:30 PM, EST

Duration:

1.5 hours

Workshop Overview:

PhysioLabXR is a Python-based application designed for visualizing, recording, and processing data streams, including making predictions. It facilitates the development of novel interaction interfaces, such as brain-computer interfaces (BCIs), and assists in conducting experiments. In this workshop, you will learn about PhysioLabXR, how to install it on your machine, and explore its key features that enable the construction of BCIs.

Target Audience:

PhysioLabXR is the primary tool for building real-time, closed-loop BCI systems during the hackathon. We recommend that all participants attend this session or view the recordings. We also encourage anyone who is interested in building BCI or VR/AR applications to participate. This workshop is not only beneficial for hackathon participants but also invaluable for those curious about the intersection of neuroscience and interactive technologies. Whether you're looking to develop applications for VR/AR environments or explore the possibilities within BCI, this session will provide you with the foundational knowledge and tools to start your journey.

Preparation Requirements:

It is strongly recommended to use PyCharm Professional as the Python IDE. You can apply for a free educational license here. Please download Python (versions 3.9 or 3.10) and clone PhysioLabXR to your computer using PyCharm.
